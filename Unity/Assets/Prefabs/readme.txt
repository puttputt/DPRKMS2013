== Fireworks package == 

Version: 1.0

Package contains:

- ten different firework effects in four different colors (blue, green, red, yellow)
- materials and textures
- Fireworks.js script

Instructions:

- for better performance, you can uncheck the variable "Enabled_light" in the Script component "Fireworks"
- to get fireworks to work, use "static function Instantiate (original : Object, position : Vector3, rotation : Quaternion) : Object"
